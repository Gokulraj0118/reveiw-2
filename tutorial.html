CSS Styling

CSS (Cascading Style Sheets) is used to style the HTML elements of your web page.

Create a new file called styles.css and link it to your HTML file:

<!DOCTYPE html>
<html>
<head>
    <title>Learning Platform Tutorial</title>
    <link rel="stylesheet" type="text/css" href="styles.css">
</head>
<body>
    <!-- Your content goes here -->
</body>
</html>
